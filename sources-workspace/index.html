Sources - Workspace

Filesystem

1. 點此下載測試頁面檔案
2. 解壓縮後在資料夾內執行 npx serve .
3. 瀏覽器開啟 localhost:5000，打開 DevTools > Sources > Filesystem
4. 將下載的資料夾拖入 Filesystem，點擊上方的「Allow」，檔名旁會有綠點
5. 修改有綠點的 index.html、style.css、index.js 並存檔就會同步變化

Overrides

1. 打開 DevTools > Sources > Overrides
2. 點擊 + Select folder for overrides，選一個資料夾來儲存改動
3. 點擊上方的「Allow」，開始編輯各個檔案
4. 存檔後看到紫色圓點，就可以重新整理頁面來看看結果
